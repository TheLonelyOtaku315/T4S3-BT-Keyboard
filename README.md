# T4-S3 Bluetooth Keyboard Bridge

ESP32-S3 project using LilyGO T4-S3 AMOLED (2.41" touch version).

## Hardware
- LilyGO T4-S3 with 2.41" AMOLED touch display
- ESP32-S3 chip
- USB-C connectivity

## Current Features
- ✅ AMOLED display working
- ✅ Touch input working
- ✅ LVGL UI framework integrated
- ✅ Serial debugging

## Setup
1. Install PlatformIO
2. Clone this repository with submodules:
```bash
   git clone --recursive https://github.com/TheLonelyOtaku315/T4S3-BT-Keyboard.git
```
3. Open in VS Code with PlatformIO
4. Build and upload
